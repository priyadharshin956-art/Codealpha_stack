📌 Project Overview
This is a simple text-based Hangman game developed in Python. The player guesses a hidden word one letter at a time. The game provides up to six incorrect attempts before the game ends.

🎯 Objective
The objective of this project is to create a console-based Hangman game that helps users improve their logical thinking and understanding of Python programming concepts.

✨ Features
- Randomly selects a word from a predefined list.
- Allows one-letter guesses.
- Maximum of 6 incorrect attempts.
- Displays the current progress of the word.
- Prevents repeated guesses.
- Shows a winning or losing message.

🛠️ Technologies Used
- Python 3

📚 Python Concepts Used
- Variables
- Lists
- Strings
- Loops ('while')
- Conditional Statements ('if-else')
- Functions
- Random Module

 ▶️ How to Run
1. Install Python 3 on your system.
2. Download or clone the project.
3. Open the project folder in a terminal or command prompt.
4. Run the following command:

bash
python hangman.py
